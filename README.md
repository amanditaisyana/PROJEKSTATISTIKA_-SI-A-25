# STATISTIK SI_A 25 AMANDITA ISYANA PUTRI_F5212510017
## Deskripsi
Proyek ini bertujuan untuk menganalisis pengaruh tingkat pendidikan terhadap Tingkat Pengangguran Terbuka (TPT) di Indonesia pada periode 2015–2025 menggunakan metode Regresi Linier Berganda. Analisis dilakukan untuk mengetahui hubungan antara beberapa variabel tingkat pendidikan dengan TPT serta membangun model prediksi berdasarkan pola data yang tersedia.

## Dataset
Dataset yang digunakan merupakan data Tingkat Pengangguran Terbuka berdasarkan tingkat pendidikan di Indonesia tahun 2015–2025.
Variabel yang digunakan meliputi:
- X1 : Tingkat Pengangguran Terbuka penduduk berpendidikan SD ke bawah
- X2 : Tingkat Pengangguran Terbuka penduduk berpendidikan SMP
- X3 : Tingkat Pengangguran Terbuka penduduk berpendidikan SMA
- X4 : Tingkat Pengangguran Terbuka penduduk berpendidikan SMK
- X5 : Tingkat Pengangguran Terbuka penduduk berpendidikan Diploma I/II/III
- X6 : Tingkat Pengangguran Terbuka penduduk berpendidikan Universitas
- Y  : Tingkat Pengangguran Terbuka (TPT) total

## Sumber data:
- Badan Pusat Statistik (BPS) Indonesia periode 2015–2025.

  ## Output yang dihasilkan
- Statistik deskriptif dataset
- Visualisasi hubungan antar variabel
- Persamaan model Regresi Linier Berganda
- Nilai koefisien regresi untuk setiap variabel pendidikan
- Hasil evaluasi model (R², MAE, dan RMSE)
- Prediksi Tingkat Pengangguran Terbuka berdasarkan tingkat pendidikan
- Interpretasi hasil analisis dan kesimpulan penelitian
